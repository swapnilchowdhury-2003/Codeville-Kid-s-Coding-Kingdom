## 🌟 What is Codeville?

Codeville is a **story-based programming learning platform** designed specifically for young children. Kids learn fundamental programming concepts through interactive stories and games - **without writing any code!**

### ✨ Key Features

- 📖 **7 Interactive Stories** - Each teaches a core programming concept
- 🎮 **Game-Based Learning** - Learn by playing, not reading
- 👤 **Personal Profiles** - Track progress with custom avatars
- 🏆 **Leaderboard & Competition** - Compete with friends
- ⭐ **Star System** - Earn up to 3 stars per story
- 🎯 **Badge Achievements** - Unlock special badges
- 📊 **Progress Dashboard** - See learning journey
- 🎨 **Colorful & Animated** - Everything bounces and moves!
- 📱 **Mobile Friendly** - Works on tablets and phones

## 🚀 Quick Start

### For Kids & Parents

1. **Open the app:**
   - Double-click `index.html` in your browser
   - Or visit the hosted version (if deployed)

2. **Create a profile:**
   - Enter your name
   - Pick a fun avatar (🐱🐶🐼🦁🐸🦄🤖🐙)
   - Choose your favorite color

3. **Start learning:**
   - Click on any story to begin
   - Complete stories to earn stars
   - Collect badges and climb the leaderboard!

### For Developers

```bash
# Clone the repository
git clone https://github.com/yourusername/codeville.git

# Navigate to the project
cd codeville

# Open in browser
# Simply open index.html in your web browser
# No build process or installation required!
```

## 🎨 Design Philosophy

### Kid-Friendly Design
- **Large Text:** Minimum 24px font size
- **Big Buttons:** At least 80px tall for easy tapping
- **Bright Colors:** Happy, vibrant color palette
- **Smooth Animations:** Everything bounces and moves
- **Touch-Friendly:** Works great on tablets

### Learning Approach
- **No Code Syntax:** Kids learn concepts, not syntax
- **Story-Based:** Every lesson is an adventure
- **Visual Learning:** See concepts in action
- **Immediate Feedback:** Instant responses to actions
- **Positive Reinforcement:** Celebrate every achievement

### What's Tracked

## 🔧 Customization

### Adding New Stories

1. Create a new HTML file in `stories/` folder
2. Follow the existing story template
3. Add story data to `js/progress.js`
4. Update dashboard to include new story

### Changing Colors

Edit `css/main.css`:
```css
:root {
  --cody-blue: #4A90E2;
  --happy-yellow: #FFD93D;
  --fun-pink: #FF6B9D;
  /* Customize these! */
}
```

### Adding Sounds

1. Add audio files to `sounds/` folder
2. Update sound references in story files
3. Ensure browser audio support

## 🐛 Troubleshooting

### Progress Not Saving?
- Check if browser allows localStorage
- Try a different browser
- Clear browser cache and try again

### Stories Not Loading?
- Ensure all files are in correct folders
- Check browser console for errors
- Verify file paths are correct

### Animations Not Working?
- Update to a modern browser
- Enable JavaScript
- Check CSS animations support
